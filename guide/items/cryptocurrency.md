# Creating cryptocurrency

To create a dynamic price in the store, you need to go to the section "Shop" -> "Change the price in the store" -> Select the price for the item (another item, currency, cookies) -> Select "Create a currency pair" -> Enter the currency pair and pricing multiplier. After that, the price for this item will be proportional to the real currency pair.
